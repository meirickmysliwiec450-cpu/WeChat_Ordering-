# 数据库初始化执行指南

## ✅ 快速开始（3步）

### 第1步：启动MySQL服务

**Windows 用户**（以管理员身份打开PowerShell）：

```powershell
net start MySQL80
```

**验证MySQL是否运行**：

```bash
mysql -u root -p
```

输入密码后，如果出现 `mysql>` 提示符，说明连接成功。

### 第2步：创建数据库

在MySQL命令行中执行：

```sql
-- 创建数据库
CREATE DATABASE wechat_ordering DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 查看数据库是否创建成功
SHOW DATABASES;
```

### 第3步：导入SQL建表脚本

**方法A：使用命令行导入（推荐）**

在 PowerShell 中执行（需要替换为您的实际路径）：

```bash
mysql -u root -p wechat_ordering < "e:\综合课程设计Ⅲ\WechatOrdering\wx-order-ssm-back\src\main\resources\db.sql"
```

**方法B：在MySQL客户端中导入**

在MySQL命令行中执行：

```sql
USE wechat_ordering;
SOURCE db.sql;
```

**方法C：在Navicat中导入**

1. 右键新建数据库 `wechat_ordering`
2. 右键数据库 → 运行SQL文件
3. 选择 `db.sql` 文件
4. 点击开始

## 🐛 常见问题排查

### 问题1：连接超时 (Connection timed out)

```
Error: connect ECONNREFUSED 127.0.0.1:3306
```

**解决**：

- 检查MySQL服务是否启动：`net start MySQL80`
- 检查端口是否正确（默认3306）
- 检查防火墙设置

### 问题2：访问被拒绝 (Access denied)

```
ERROR 1045 (28000): Access denied for user 'root'@'localhost'
```

**解决**：

- 确认用户名和密码正确
- 检查 `application.properties` 中的密码配置

### 问题3：字符编码错误

```
Incorrect string value: '\xF0\x9F\x8D...' for column...
```

**解决**：

- 确保SQL中有 `DEFAULT CHARSET=utf8mb4`
- 数据库连接URL中包含 `characterEncoding=utf8mb4`

### 问题4：表已存在错误

```
ERROR 1050 (42S01): Table 'xxx' already exists
```

**解决**：

- 删除已存在的数据库后重新创建

```sql
DROP DATABASE IF EXISTS wechat_ordering;
CREATE DATABASE wechat_ordering DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 📌 常用SQL操作

### 查看所有表

```sql
USE wechat_ordering;
SELECT TABLE_NAME FROM information_schema.TABLES WHERE TABLE_SCHEMA = 'wechat_ordering';
```

### 删除并重建数据库

```sql
DROP DATABASE IF EXISTS wechat_ordering;
CREATE DATABASE wechat_ordering DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE wechat_ordering;
SOURCE db.sql;
```

### 查看表占用空间

```sql
SELECT
  TABLE_NAME,
  ROUND(((data_length + index_length) / 1024 / 1024), 2) AS size_mb
FROM information_schema.tables
WHERE table_schema = 'wechat_ordering'
ORDER BY (data_length + index_length) DESC;
```

### 备份数据库

```bash
mysqldump -u root -p wechat_ordering > wechat_ordering_backup.sql
```

### 恢复数据库

```bash
mysql -u root -p wechat_ordering < wechat_ordering_backup.sql
```
