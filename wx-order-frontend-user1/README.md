# 微信点餐系统 - 商家管理后台

基于 Vue 3 + Vite + Element Plus 的微信点餐系统商家端管理后台。

## 功能模块

- **仪表盘** - 数据概览：用户总数、订单统计、营业额、菜品数量
- **用户管理** - 查看用户列表、启用/禁用用户账号
- **分类管理** - 菜品分类的新增、编辑、删除、排序
- **菜品管理** - 菜品的新增、编辑、上下架、删除，支持按分类筛选和关键词搜索
- **订单管理** - 订单列表查看、按状态筛选、接单/拒单/完成操作、订单详情
- **轮播图管理** - 首页轮播图的新增、编辑、启用/禁用、删除、排序
- **反馈管理** - 用户反馈列表、管理员回复、标记已解决

## 技术栈

- Vue 3 (Composition API)
- Vite 构建工具
- Element Plus UI 组件库
- Vue Router 路由管理
- Pinia 状态管理
- Axios HTTP 请求库

## 项目启动

```sh
# 安装依赖
npm install

# 开发模式运行（热更新）
npm run dev

# 生产环境构建
npm run build
```

## 后端对接

后端 Spring Boot 服务运行在 `http://localhost:8080/api`，前端通过 Vite 代理或直接请求后端接口。

默认管理员账号：`admin` / `123456`
